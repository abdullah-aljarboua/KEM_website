KAPSARC Energy Model

Execution

Here are the prerequisites for KEM, and the steps to download and execute the model.

Prerequisites

  • Windows environment
  • GAMS 24.1.3 or higher
  • GAMS license for the solver

Installing & Executing

$ git clone https://github.com/KAPSARC/KEM_EMP.git
$ cd KEM_EMP
$ KEM_EMP.bat
